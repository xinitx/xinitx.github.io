图形结构

Posted on 2021-12-02 Edited on 2022-02-08 In Algorithm , 数据结构

前言

由顶点和连接顶点的边构成的离散结构

分类

  1. 有向图：边集具有方向的图，无向图：特殊的有向图、每一条边都是对称的
  2. 边相对顶点的数量：稠密图、稀疏图（任意两个顶点都有边的图称完全图）
  3. 加权图：每一条边具有相应的权重
  4. 连通图：任意两个顶点都是连通的
  5. 强连通图：任意两个顶点是双向连通的

表示方法

邻接矩阵

第一维存放图中所有顶点数据，用一个二维数组存放顶点间关系（边或弧）的数据。

  1. 适用于稠密图
  2. 可存有向图、无向图
  3. 空间复杂度：O(n^2)
  4. 无法存重边

邻接表

第一维描述某个点，第二维描述这个点所对应的边集们

  1. 适用于稀疏图
  2. 可存有向图、无向图
  3. 空间复杂度：O(n + m)
  4. 可存重边

逆邻接表

邻接表描述的是出边，逆邻接表描述的是入边

邻接多重表

邻接表的优化，所有边按两端顶点链接在一起

  1. 适用于稀疏图
  2. 可存无向图
  3. 空间复杂度：O(n + m)
  4. 可存重边

十字链表

对邻接矩阵的优化，可以看成是将有向图的邻接表和逆邻接表结合起来得到的，即每一个顶点指向入边和出边

  1. 适用于稀疏图
  2. 可存有向图、无向图
  3. 空间复杂度：O(n + m)
  4. 无法存重边

三元组表

  1. 适用于稀疏图
  2. 可存有向图，无向图
  3. 常用于Bellman-Ford算法、Kruskal算法
  4. 空间复杂度：O(m)
  5. 可存重边
