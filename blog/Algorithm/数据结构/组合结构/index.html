组合结构

Posted on 2021-12-02 Edited on 2022-02-08 In Algorithm , 数据结构

前言

数据结构中的元素之间除了“同属一个集合” 的相互关系外，别无其他关系

哈希表

哈希表（Hash table，也叫散列表），是根据关键码值(Key value)而直接进行访问的数据结构。

  1. 散列函数：关键字为k且其值存放在f(k)的存储位置上，f(k)即散列函数
  2. 均匀散列函数：关键字集合中的任一个关键字，经散列函数映象到地址集合中任何一个地址的概率是相等的
  3. 冲突：不同的关键字可能得到同一散列地址

构建方法

拉链法

邻接表类似，每一个节点对应一条链表

开放寻址法

占坑法，比拉链法时间复杂度高

STL

pair

tuple

可以封装不同类型的任意数量的对象，可以和pair相互构造。但是，其成员数量不确定，所以没有成员名

map

基于pair，键的值不能重复和修改,会根据键大小自动排序

multimap

类似map，但是它可以同时存储多个键相同的值，没有at()方法，没有重载 [] 运算符，所以取值只能遍历

unordered_map

无序map，相比map仅无序这一点不同，少四个逆向迭代器，少lower_bound(key)、upper_bound(key)

unordered_mutimap

无序mutimap，相比unordered_map可以同时存储多个键相同的值，没有at()方法，没有重载 [] 运算符，所以取值只能遍历

set

自动排序，基于pair且键和值必须相等

multiset

相比set 容器不同的是，multiset 容器可以存储多个值相同的元素，但是删除时会删除所有相同值

unordered_set

相比set 容器，不会再以键值对形式存储数据，不会对内部存储的数据排序，少四个逆向迭代器，少lower_bound(key)、upper_bound(key)

unordered_multiset

相比unordered_set 容器不同的是，unordered_multiset 容器可以存储多个值相同的元素，但是删除时会删除所有相同值
