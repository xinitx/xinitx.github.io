线性结构

Posted on 2021-12-01 Edited on 2022-02-08 In Algorithm , 数据结构

前言

数据结构中的元素存在一对一的相互关系即线性结构

线性列表

n个具有相同特性的数据元素的有限序列，逻辑上相邻元素之间存在着一一对应关系，所以堆不是线性结构。

按逻辑层次划分
对结点的操作是否受限制

  1. 一般线性表：数组、链表
  2. 受限线性表：栈、队列

按存储结构划分
存储单元的连续性

  1. 顺序表示：数组、栈、队列
  2. 链式表示：链表

数组

字符数组

在翻译阶段7，由一个或多个串字面值或字面值得到的每个多字节字符序列的末尾添加一个值为0的字节或代码（C标准）

string

string 的结尾没有结束标志’\0’，所以不能直接使用string.h的函数，但是可以使用c_str()成员函数进行转换

STL array

数组，大小是固定的，比普通数组更安全，效率并没有因此变差。

STL vector

矢量

栈

STL stack
数据结构栈又称堆栈

  1. push 入栈
  2. pop 出栈
  3. empty 没有元素时返回true
  4. emplace 用传入的参数调用T的构造函数，并将对象入栈，比如push(data(1, 2)) == emplace(1, 2)
  5. size 大小
  6. swap 当前栈所有元素与参数栈所有元素交换，两者必须类型相同。全局swap()也可
  7. top 栈顶元素
  8. 具有=和比较运算符，按字典比较

单调栈

队列

STL queue

  1. front() 队头
  2. back() 队尾
  3. push() 入队
  4. pop() 出队
  5. size() 大小
  6. empty() 没有元素时返回true
  7. emplace() 用传入的参数调用T的构造函数，并将对象入队
  8. swap() 将当前 queue 中的元素和参数 queue 中的元素交换，两者必须类型相同。全局swap()也可
  9. 具有=和比较运算符，按字典比较

单调队列

双向队列

STL deque
包括vector类似的构造函数和成员函数，不同点主要有

  • 随机存取慢于vector
  • vector在尾部增删元素性能好，deque在头部和尾部增删元素性能好
  • 比vector多两个成员函数：push_front，pop_front

双向链表

不支持下标随机存取，双向链表的每个元素中都有一个指针指向后一个元素，也有一个指针指向前一个元素

STL list
包括vector类似的构造函数和成员函数，不同点主要有

  • push_front()，pop_front()
  • sort
  • remove(buf)，删除和buf相等的元素
  • remove_if，删除符合某种条件的元素
  • unique()，删除所有和前一个元素相等的元素
  • merge(list x)，将x合并，清空x
  • splice(iterator i，list x，iterator begin， iterator end) 在i前面插入x的[begin, end)。当i不在[begin, end)区间内，可以是同一个列表。

单链表

能用单链表的地方就能用双链表，但是单链表空间消耗和执行效率更高

STL forward_list
包括list类似的构造函数，不同点主要有

  • 没有反向迭代器，有before_begin()和cbefore_begin()指向容器第一个元素之前的位置
  • emplace_front() 在容器头部生成一个元素。该函数和 push_front() 的功能相同，但效率更高。
  • emplace_after() 在指定位置之后插入一个新元素，并返回一个指向新元素的迭代器。和 insert_after() 的功能相同，但效率更高。
  • insert_after() 在指定位置之后插入一个新元素，并返回一个指向新元素的迭代器。
  • erase_after() 删除容器中某个指定位置或区域内的所有元素。
  • splice_after() 将某个 forward_list 容器中指定位置或区域内的元素插入到另一个容器的指定位置之后。
  • reverse() 反转容器中元素的顺序。

STL相关

容器适配器
把一种容器转换成另一种容器，实际上stack、queue、priority_queue是通过其他容器实现的
容器适配器 基础容器筛选条件 默认使用的基础容器
stack 基础容器需包含以下成员函数：
  • empty()
  • size()
  • back()
  • push_back()
  • pop_back()
满足条件的基础容器有 vector、deque、list。		 deque
queue 基础容器需包含以下成员函数：
  • empty()
  • size()
  • front()
  • back()
  • push_back()
  • pop_front()
满足条件的基础容器有 deque、list。		 deque
priority_queue 基础容器需包含以下成员函数：
  • empty()
  • size()
  • front()
  • push_back()
  • pop_back()
满足条件的基础容器有vector、deque。		 vector

函数

  • find(begin，end，target)，需要迭代器，适用于所有的序列式容器。底层实现，其实就是用==运算符逐一比对。
  • sort(begin，end)
  • begin()、end()，可用于普通数组
